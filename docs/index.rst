Status & License
================

Lilac is distributed schedule system wrote by python, It was originally
written and maintained by `Thomas <https://github.com/thomashuang>`_.


lilac is released under an `GPL license <license.html>`_.


lilac development happens on `GitHub <https://github.com/thomashuang/Lilac/tree/master>`_.
Development version is installable via `easy_install
lilac==dev <https://github.com/thomashuang/lilac>`__.  You
can clone the source code with::

    $ git clone https://github.com/thomashuang/Lilac.git
